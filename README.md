# Marketing-Funnel-Conversion-Performance-Analysis
Marketing Funnel Analysis using Python
## 🔍 Objective
Analyze customer journey from leads to conversion using Bank Marketing Dataset.

## 📁 Dataset
Bank Marketing Dataset (UCI)

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📈 Funnel Stages
- Total Customers
- Contacted
- Interested Leads
- Converted Customers

## 📊 Key Insights
- High drop-off at conversion stage
- Campaign overuse reduces performance
- Channel-wise conversion differs

## 💡 Recommendations
- Improve closing strategy
- Focus on high-performing channels
- Optimize campaign frequency
